# 🛒 Amazon Clone

A responsive Amazon-inspired e-commerce website built with **HTML, CSS, and JavaScript**. A responsive Amazon-inspired e-commerce website featuring product browsing, cart management, quantity updates, and dynamic order summaries.

- 📦 Product listing with images, ratings, and prices
- 🛒 Add products to cart
- 🔄 Update item quantities
- ❌ Remove items from cart
- 💰 Dynamic order summary and price calculations
- 💾 Persistent cart storage using Local Storage
- 📱 Responsive design for different screen sizes
- ⚡ Modular JavaScript using ES6 modules

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ReeSupp/Amazon-Clone.git 
   ```

2. Navigate to the project folder:
   ```bash
   cd Amazon-clone
   ```

3. Open `amazon.html` in your browser.
